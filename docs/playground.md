# Playground

Try vibe-flags right here — no install, no build step. Edit the HTML on the left and see it render live on the right.

<Playground />

## How to use

1. Pick a preset or write your own HTML in the editor.
2. The preview updates automatically as you type (300 ms debounce).
3. Use the toolbar button on the right edge of the preview to open the flag panel and toggle flags.
